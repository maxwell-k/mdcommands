# mdcommands

This repository contains two command line utilities for working with markdown:

1. [mdcommands](./mdcommands/README.md)
2. [dislodge](./dislodge/README.md)

## Dependencies

- [Deno](https://deno.com/)

For ZSH integration:

- [fzf](https://junegunn.github.io/fzf/)
- [zsh](https://www.zsh.org/)
